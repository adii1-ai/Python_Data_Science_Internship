Commands :-

two methods to push local repo on Github
    1. Github --->>> local
    2. Local --->>> Github

A] steps for Method 1 (Github --->>> local):

1. Create new repository on Github
2. Copy the URLof this repo
3. Create Folder on Desktop 
4. Open This Folder on Visual Studio Code Editor.
5. Open Terminal and Enter Following commands:

    -git clone URL_of_Repo
6. Create/ Add/ Write new Files in VS Code inside this repo.
    -git status   # to check the which files Updated/ Modified but not commited and pushed
    -git add .   # to add all the newly created/ Updated Files
    -git commit -m "Any appropriate message inside this quote's"
    -git push origin main    # this command push the files on github
